# Gateway Features

Discover and configure Merge Gateway's advanced features: tags, project tracking, routing metadata, extended thinking, and multi-model routing.

## When to Use

After initial Gateway integration, when users want to:
- Organize API calls with tags and project IDs
- Understand routing decisions and cost breakdowns
- Use extended thinking (Claude models)
- Route requests across multiple providers through a single client
- Monitor usage and optimize costs

## Prerequisites

- Merge Gateway SDK installed (`pip install merge-gateway-sdk`)
- `MERGE_GATEWAY_API_KEY` and `MERGE_GATEWAY_BASE_URL` configured
- Basic integration working (run `gateway-implement` first)
